# 알림톡 템플릿 (카카오 사전심사용 문안)

아래 문안을 **대행사 콘솔(솔라피)의 알림톡 템플릿 등록**에 그대로 넣고 심사 요청합니다.
심사 통과 후 발급되는 **템플릿 ID**를 환경변수(`SOLAPI_TEMPLATE_SIGN` / `SOLAPI_TEMPLATE_DONE`)에 넣으면
서버가 그 승인본으로만 발송합니다.

> 원칙: 알림톡은 **정보성 메시지**여야 합니다(광고·홍보 문구 금지). 아래 문안은 고객이 요청·진행 중인
> "계약" 거래에 대한 안내라 정보성 요건에 맞습니다. 변수는 `#{...}` 형식이며, 서버가 채웁니다.

---

## 1) `contract_sign` — 전자계약 서명 요청

**카테고리(예시):** 서비스 이용 > 계약/신청
**강조표기(선택):** 전자계약 서명 요청

```
[만물인테리어] 전자계약 서명 요청

#{name}님, 안녕하세요.
만물인테리어에서 요청하신 공사 계약서를 보내드립니다.

▶ 계약번호: #{contractNo}
▶ 현장: #{site}
▶ 계약금액: #{amount}원

아래 버튼을 눌러 본인확인 후 계약 내용을 확인하고 서명해 주세요.
※ 서명 링크는 72시간 후 만료됩니다.
```

**버튼:** 웹링크(WL) — 이름 `계약서 확인·서명`, 모바일/PC URL `#{signUrl}`

**변수 채움(서버):** `name`, `contractNo`, `site`, `amount`, `signUrl`

---

## 2) `contract_done` — 전자계약 완료

**카테고리(예시):** 서비스 이용 > 계약/신청
**강조표기(선택):** 전자계약 완료

```
[만물인테리어] 전자계약 완료

#{name}님, 전자서명이 정상적으로 완료되었습니다.

▶ 계약번호: #{contractNo}
▶ 완료일시: #{completedAt}

계약서 사본은 아래 버튼에서 확인하실 수 있습니다.
(보안을 위해 확인 링크는 15분간만 유효합니다)
```

**버튼:** 웹링크(WL) — 이름 `계약서 사본 보기`, 모바일/PC URL `#{viewUrl}`

**변수 채움(서버):** `name`, `contractNo`, `completedAt`, `viewUrl`

---

## 3) `contract_invoice` — 공사대금 안내

**카테고리(예시):** 서비스 이용 > 결제/청구
**강조표기(선택):** 공사대금 안내

```
[만물인테리어] #{name}님, #{stageLabel} 안내드립니다.
계약번호: #{contractNo}
#{stageLabel}: #{amount}원
입금계좌: #{payInfo}
확인 후 입금 부탁드립니다. 감사합니다.
```

**변수 채움(서버):** `name`, `stageLabel`(계약금/중도금/잔금), `contractNo`, `amount`, `payInfo`(계좌)
버튼 없음(정보성 안내). 재발송은 '독촉'으로 기록됨.

---

## 심사 시 유의(반려 예방)

- 변수만으로 이루어진 문장 금지 → 위처럼 고정 안내문 + 변수 조합이어야 함.
- 광고성 표현(할인·이벤트·"지금 상담") 금지.
- 버튼 URL은 변수(`#{signUrl}`)로 두되, 실제 도메인은 발송 시 서버가 채운 값이 됨(HTTPS 필수).
- 발신프로필(카카오 채널)과 사업자 정보가 일치해야 함(만물인테리어 / 895-48-01132).
- 대체발송(문자) 문안이 필요하면 대행사 콘솔에서 별도 등록(알림톡 실패 시 자동 대체).
