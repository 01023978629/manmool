# 만물인테리어 — 사장님 인계서 (AI → 사람)

> **한 줄 요약:** 코드·테스트는 전부 끝났고, 시스템 전체가 지금 **"안전 모드(off)"** 입니다.
> 실제 카톡 발송·자동 청구·리드 전송이 하나도 켜져 있지 않아, 켜기 전엔 아무 일도 일어나지 않습니다.
> 아래 **4가지**만 사장님 계정·명의로 켜시면 가동됩니다.

**왜 이건 사장님만 할 수 있나요?**
호스팅 계정, 카카오 채널 명의, 전자서명 같은 **"사장님 신원"** 이 필요한 단계라 AI가 대신할 수 없습니다.
(판단·문서·데이터 정리는 AI가 이미 끝냈고, 여기부터가 "몸은 사람" 영역입니다.)

---

## ✅ 사장님이 할 일 (우선순위 순)

### [ ] 1순위 — 전자계약 서버 배포 ⭐ 가장 중요
고객이 카톡으로 받은 **서명 링크를 폰에서 열려면** 서버가 공개 주소에 떠 있어야 합니다.
이게 없으면 전자계약 전체가 작동하지 않습니다.

- **준비물:** 호스팅 계정 1개(Fly.io 권장), 영속 디스크(볼륨), 시크릿 2개
- **시크릿 만들기** (터미널에 그대로 입력):
  ```
  openssl rand -hex 32      ← CONTRACT_PEPPER 값
  openssl rand -hex 16      ← ADMIN_TOKEN 값 (관리자 로그인용 · 따로 메모)
  ```
- **절차:** `contract-backend/DEPLOY.md` 의 A(Fly.io) 몇 줄 그대로 따라 하기
- **마지막 확인:** 배포 환경에서 `npm run preflight` → **초록불(✅ 배포 준비 완료)** 확인 후 가동
- ⏱ 30분 내외 · 💰 소액(월 몇 천 원대)

### [ ] 2순위 — 알림톡 실발송 켜기
실제 **카카오톡** 으로 계약·청구 문자를 보내려면 필요합니다.
(안 켜면 전부 Mock — 문안·흐름은 다 되지만 실제 발송만 안 됩니다.)

- 카카오 비즈니스 채널 개설 + 솔라피(Solapi) 가입 + **템플릿 승인**(카카오 심사 며칠 소요)
- 배포된 `https://<서버주소>/admin` 접속 → 값 입력 → **실발송 = 1** → 본인 번호로 테스트 발송
- **안내 문서:** `contract-backend/SETUP-ALIMTALK.md`

### [ ] 3순위 — 홈페이지 상담 폼 자동 전달 켜기
지금은 데모라 **폼은 뜨지만 신청이 자동으로 사장님께 오지 않습니다**
(현재는 고객이 전화·문자·현장앱 링크로만 옵니다).
켜면 상담 신청이 **이메일이나 카톡으로 즉시** 도착합니다.

- 무료 방법: Web3Forms(무료) 또는 n8n → `data/config.json` 에서 활성화
- **안내 문서:** `integrations/SETUP-n8n-kakao.md`
- ⏱ 15분 · 💰 무료

### [ ] 현장앱(hyeonjang) — 폰에 설치
- 폰 브라우저로 앱 주소 열기 → **홈 화면에 추가**(앱처럼 사용)
- (선택) 구글 드라이브 연결 → 사진·현장 데이터 클라우드 자동 백업
